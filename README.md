# Project Phoenix Test 

**Backend (Node.js / Express)**
* TypeScript & tsx, Zod, Luxon, Pino, Jest 

**Mobile (React Native / Expo SDK 54)**
* React Query, Styled-Components, React Navigation

**CI/CD**
* Docker, Github Actions

~6 hours (4 backend, 1 mobile, 1 ci/cd)

---

## Run the Project

### 1. Backend
runs on port 8080 

```bash
cd backend
npm i

npm run test
npm run dev
```

### 2. Mobile
```
cd mobile
npm i
npx expo prebuild

# Start and use Expo Go from iphone (SDK 54)
npx expo start
```
